Save these files inside your project folder:

C:\Users\Thera\therassistant ehr

After unzipping, you should have:
- C:\Users\Thera\therassistant ehr\scripts\restart-backend.ps1
- C:\Users\Thera\therassistant ehr\scripts\restart-frontend.ps1
- C:\Users\Thera\therassistant ehr\scripts\restart-both.ps1
- C:\Users\Thera\therassistant ehr\.vscode\tasks.json

In VS Code:
1. Open the folder: C:\Users\Thera\therassistant ehr
2. Press Ctrl+Shift+P
3. Run: Tasks: Run Task
4. Choose:
   - Restart Backend
   - Restart Frontend
   - Restart Both

The easiest option is "Restart Both".
